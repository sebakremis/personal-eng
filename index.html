The AI Divergence: Visualizing NVIDIA's Fundamentals vs. Price Action

One of the biggest challenges in analyzing high-growth stocks like NVIDIA (NVDA) is contextualizing the stock price explosion against actual business performance. Using a custom Galaxy Schema in Power BI, I merged daily price action with SEC filing data to visualize this correlation at two distinct levels of granularity.

1. The Macro View (5-Year Trend)

First, I established the long-term baseline. This chart combines 5 years of daily price closes with annual revenue totals. Note the "Backfilled" data for 2020-2021, which I manually sourced from SEC filings to bridge the gap in the yfinance API.

2. The Micro View (2025 Drill-Down)

Using Power BI's hierarchy features, I created a drill-down capability to inspect the current fiscal year. This view reveals the quarterly volatility often hidden in annual aggregates.

Technical Implementation

  • Data Modeling: Implemented a Galaxy Schema with a shared Date Dimension to link daily Price facts with quarterly Financial facts.
  • DAX Logic: Used ISINSCOPE to create dynamic measures that switch calculation logic based on the user's drill-down level (Year vs. Quarter).
  • Data Engineering: Python ETL pipeline extracts raw market data, enforces a 5-year rolling window, and normalizes it into Parquet files for Power BI ingestion.

Data Sources: Market data via Yahoo Finance API (Python/yfinance). Historical fundamentals for 2020-2021 sourced directly from SEC Filings. ETL Pipeline built with Python; Modeling in Power BI.
